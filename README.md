A clean time and date display for your UI. Shows Local time, Server time, and Date with smart formatting and dynamic sizing. Fully configurable appearance and layout.

Features
- Local/Server time with 12/24h and optional seconds
- Date with day suffix (e.g., 21st) and optional month color themes
- Vertical or horizontal layout with adjustable spacing
- Custom font, size, and outline (LibSharedMedia-3.0)
- Backdrop colors, opacity, and border toggle
- Minimap button (LibDataBroker-1.1 + LibDBIcon-1.0):
  - Left-click: Lock/Unlock the frame
  - Right-click: Open settings
- Per-profile settings and a Profiles tab (AceDBOptions)

Configuration
- Type /mytd or /mytdconfig to open settings
- Layout tab: horizontal/vertical, text alignment, spacing controls
- Backdrop tab: background and border colors, opacity, show border
- Appearance tab: font, size, outline
- Minimap tab: show/hide minimap button
- Advanced tab: reset to defaults

Optional Dependencies
- LibSharedMedia-3.0 (fonts and textures)
- LibDataBroker-1.1 and LibDBIcon-1.0 (minimap button)

Notes
- Drag the frame to reposition (disable when locked)
- Settings are profile-based and persist across sessions
